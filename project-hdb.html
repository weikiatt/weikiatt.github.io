Predicting HDB Flat Prices (BT4240)

The Problem

Navigating the Singapore public housing market is challenging for both buyers and sellers. Determining a fair resale price involves considering a multitude of variables including flat characteristics, location, and proximity to crucial amenities.

The Solution

I developed a predictive model to estimate HDB resale flat prices accurately. By utilizing neighborhood and amenity information alongside core flat attributes, the model provides actionable pricing insights to empower stakeholders in the housing market.

Methodology & Tech Stack

  • Data Engineering with APIs: Leveraged the OneMap API to creatively engineer distance-related features, such as calculating the exact proximity to the nearest MRT station for each flat.
  • Model Optimization: Utilized hyperparameter grid search and various optimization techniques to enhance model accuracy.
  • Model Evaluation: Systematically trained and evaluated multiple algorithms including Linear Regression, Support Vector Machines (SVM), Random Forests, and Neural Networks to identify the optimal predictive model.
