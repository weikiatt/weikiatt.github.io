NUS Orbital

The Problem

University students often find information highly fragmented across different modules, courses, and informal chats. There is a need for an inclusive, centralized space where students can actively engage with peers, ask questions, and share knowledge related to their coursework.

The Solution

As part of the School of Computing's self-driven programming summer experience, I developed a comprehensive student forum website. The platform is designed to foster collaborative learning, functioning similarly to Reddit or Piazza but tailored specifically for the NUS community.

Technology & Implementation

  • Full-Stack Development: Utilized C#, JavaScript, HTML, and CSS to build a responsive and dynamic forum interface.
  • Cloud Deployment: Deployed the application using Microsoft Azure to ensure high availability and scalability for the student body.
  • Machine Learning Integration: Addressed classification challenges within the forum content by training Logistic Regression, Random Forest, SVM, and GBT models, handling data imbalances with SMOTE and ENN.
