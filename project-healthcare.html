Healthcare Capstone Project

The Problem

Clinicians often need to extract insights from complex healthcare databases like OMOP CDM, but doing so requires writing complex SQL queries, which is a specialized technical skill. Furthermore, any automated solution must strictly adhere to data privacy laws and clinical safety standards, meaning patient data cannot leave secure environments, and AI-generated results cannot be blindly trusted.

The Solution

I architected a secure LangGraph multi-agent system that runs entirely offline via Ollama. This ensures that no sensitive patient data ever leaves the local environment. The system translates natural language queries from clinicians into dynamic SQL queries to retrieve data from the OMOP CDM.

Key Features & Outcomes

  • Natural Language to SQL: Specialized AI agents dynamically generate SQL to retrieve the necessary data.
  • Automated Statistical Analysis: The retrieved data is automatically analyzed, streamlining the research process.
  • Human-in-the-Loop (HITL) Workflow: Designed with auditability in mind, the system requires clinician validation at three review checkpoints using LangGraph interrupts, ensuring that AI outputs are reviewed before delivering final results.
